# Average Calculator

Эта программа на Python включает в себя класс AverageCalculator, который вычисляет среднее значение списка чисел и сравнивает средние значения двух списков.

## Running Tests

Для запуска тестов убедитесь, что у вас установлены pytest и pytest-cov. Вы можете установить их с помощью следующей команды:

```
pip install pytest pytest-cov
```

После установки зависимостей выполните тесты с помощью следующей команды:

```
pytest --cov=average_calculator test_average_calculator.py
```
## Проверка качества кода с помощью Pylint

Для проверки качества кода с использованием pylint убедитесь, что у вас установлен pylint. Вы можете установить его с помощью следующей команды:

```
pip install pylint
```

После установки pylint проверьте качество кода с помощью следующей команды:

```
pylint average_calculator.py
```

Эта команда проанализирует код и предоставит обратную связь по качеству кода.

## Отчет о покрытии тестами

После выполнения тестов с использованием pytest-cov вы можете найти отчет о покрытии в терминале. Кроме того, вы можете сгенерировать отчет о покрытии в формате HTML с помощью следующей команды:

```
pytest --cov=average_calculator --cov-report html test_average_calculator.py
```

Эта команда создаст отчет в формате HTML в директории htmlcov. Откройте файл index.html в вашем браузере, чтобы просмотреть подробный отчет о покрытии кода.

Вы также можете открыть html файл `...\pythonAvgApp\htmlcov\index.html`
